# Claude.MD

## Project Summary
SAM's Abidjan Website is a static, GitHub Pages-based restaurant site designed for easy no-code editing via a dedicated admin panel. It includes support for site settings (contact info, hours), menu management (dishes, prices, photos), customer reviews, and a gallery.

## What the repository contains
- `index.html`: public site landing page
- `admin/index.html`: admin panel interface
- `_data/settings.json`: site configuration values
- `admin/config.yml`: admin panel settings
dependent on static site content and assets in `/images`

## Primary user flows
1. Owner edits restaurant data in the admin panel.
2. Admin writes menu items, uploads photos, and stores ratings.
3. Site content updates are reflected on the public page via GitHub Pages.

## Goal
Make restaurant content management simple for non-developers and enable fast updates to menu/services in Abidjan.
