# SAM's Abidjan Website

## How to edit your website (no code needed!)

### Admin Panel
Go to: `https://ofiliyoungyz-spec.github.io/samsabidjan/admin`

### What you can edit:
- ⚙️ **Site Settings** — phone, address, hours
- 🍔 **Menu Items** — add/edit/remove dishes, upload photos, set prices
- ⭐ **Reviews** — add customer reviews
- 📸 **Gallery** — upload your own food photos

### To upload a photo:
1. Go to the admin panel
2. Click the section you want to edit
3. Click the image field → upload from your computer
4. Save — done!
